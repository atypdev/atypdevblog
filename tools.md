## Setup Scripts

From [this](https://github.com/atypdev/Setup) repo (AUDIT BEFORE RUNNING ALWAYS!!!)

### Ubuntu 22.04 Server:
```
sudo apt install -y git
git clone https://github.com/atypdev/Setup
cd Setup
chmod +x ServerSetup.sh
./ServerSetup.sh
```
### Windows:
```
git clone https://github.com/atypdev/Setup
cd Setup
Set-ExecutionPolicy Bypass
.\WinSetup.ps1
```
